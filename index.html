This project is a website that can help customers to make a reservation at a nail art salon by online booking and help the salon manage the customer’s list and customize its salon profile pages so that the customer could decide which salon they will choose to make a reservation with. This website was made by a team consisting of three members; I was in charge of developing the front end. This project is a website that has two kinds of interfaces, such as salon interface and customer interface, with a total of fourteen projects from the salon interface and 16 pages from the customer interface. The salon interface is desktop-based screen, while the customer interface is mobile-based screen. We made the designs have slightly different themes of color to distinguish each type of interface. The design of the interface was also different; we made sure that the design for the mobile-based interface is more mobile-user friendly and the desktop-based interface is more desktop-user friendly. When implementing the design from Figma to code, I experienced the struggle of using the wrong methods in styling the website; I used float instead of just using the display properties. I realized that my decisions were wrong when I nearly finished the page, but I didn’t want any unwanted problems to happen in the future, so I tried to use a different method instead, such as the display properties. The other struggle that I have faced is that when my teammate who was in charge of making the website’s design in Figma wanted to make a salon profile design that was similar to an Instagram profile design, then I realized that I just needed to use a display “grid” property to make it. I also tried to implement some Bootstrap templates, such as carousel. In this project, I learned more deeply about how to make a website become more interactive using JavaScript. The hardest part is to make a calendar that could show the current date and every date in the other months and other years. By using a few functions in JavaScript to generate the calendar, I could overcome the problems. JavaScript could also be used to open and close pop-out items. Building an insensitive search engine is easier by using JavaScript. The idea is to change the keyword and the items inside the database to lowercase, then use the linear search using the foreach items to search it one-by-one, then display the match. This website also made me try to widen my knowledge and realize a lot of easier and more effective ways to build a website.
